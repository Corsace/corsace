## Nilai Parameter Config yang Relevan

Berdasarkan nilai parameter config yang telah kamu tetapkan pada `config.docs.host` dan `config.docs.port` milik situs lokalmu, kamu akan dapat mengakses Corsace secara lokal pada `http://host:port`.

Sebagai contoh, apabila `config.docs.host` kamu diatur ke `localhost` dan nilai `config.docs.port` kamu adalah `10000`, maka situs lokalmu akan dapat diakses pada `http://localhost:10000`.

Nilai parameter `config.docs.publicUrl` di sini harus merupakan gabungan antara nilai `config.docs.host` dan `config.docs.port`. Pada contoh di atas, nilai ini adalah `http://localhost:10000`.

Lihat halaman [Referensi Config](../../../misc/reference/config) untuk informasi lebih lanjut.

## Menjalankan Lingkungan Sistem

Jalankan perintah berikut untuk memulai situs web dalam mode pengembangan:

```bash
npm run docs-dev
```

Situs web lokalmu akan tersedia pada URL yang kamu atur pada [Referensi Config](../../../misc/reference/config).
