## Nilai Parameter Config yang Relevan

Based on what config values you had provided for the website's `config.[WEBSITE].host` and `config.[WEBSITE].port`, you should be able to access the website at `http://[config.[WEBSITE].host]:[config.[WEBSITE].port]`. **Pastikan kamu menggunakan port yang berbeda untuk masing-masing situs web**.

Sebagai contoh, apabila `config.ayim.host` kamu diatur ke `localhost` dan nilai `config.ayim.port` kamu adalah `3000`, maka situs lokalmu akan dapat diakses pada `http://localhost:3000`.

Nilai parameter `config.[WEBSITE].publicUrl` di sini harus merupakan gabungan antara nilai `config.[WEBSITE].host` dan `config.[WEBSITE].port`. Pada contoh di atas, nilai ini adalah `http://localhost:3000`.

Lihat halaman [Referensi Config](../../../misc/reference/config) untuk informasi lebih lanjut.

## Menjalankan Lingkungan Sistem

:::note
Pastikan untuk menjalankan server API apabila server ini belum aktif. See the [Server + CrunRunner - Running Environment](../../../server/guide/server-running-environment) guide for more information.
:::

:::danger
Untuk menggunakan layanan entri masuk milik situs web, kamu akan **PERLU** untuk menjalankan situs web Main bersamaan dengan server API dan situs web yang sedang kamu kerjakan.
:::

Jalankan masing-masing perintah berikut pada terminal yang terpisah sesuai dengan situs web yang ingin kamu jalankan:

| Situs  | Cmd                    |
| ------ | ---------------------- |
| AYIM   | `npm run ayim-dev`     |
| Closed | `npm run closed-dev`\* |
| MCA    | `npm run mca-dev`      |
| Utama  | `npm run main-dev`     |
| Open   | `npm run open-dev`     |

\*Saat ini belum tersedia

Situs web lokalmu akan dapat ditemui pada alamat URL yang kamu tentukan pada [Referensi Config](../../../misc/reference/config).

:::caution
If you are using `localhost` or `127.0.0.1` as the host, and you are unable to see the website after running the command, you may need to check your hosts file to make sure they are connected (this may be necessary in Linux if not done already).
:::
