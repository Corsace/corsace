## Nilai Parameter Config yang Relevan

Berdasarkan nilai parameter config yang telah kamu tetapkan pada `config.[WEBSITE].host` dan `config.[WEBSITE].port` milik situs lokalmu, kamu akan dapat mengakses Corsace secara lokal pada `http://host:port`.

Sebagai contoh, apabila `config.ayim.host` kamu diatur ke `localhost` dan nilai `config.ayim.port` kamu adalah `3000`, maka situs lokalmu akan dapat diakses pada `http://localhost:3000`.

Nilai parameter `config.[WEBSITE].publicUrl` di sini harus merupakan gabungan antara nilai `config.[WEBSITE].host` dan `config.[WEBSITE].port`. Pada contoh di atas, nilai ini adalah `http://localhost:3000`.

Lihat halaman [Referensi Config](../../../misc/reference/config) untuk informasi lebih lanjut.

## Menjalankan Lingkungan Sistem

:::note
Pastikan untuk menjalankan server API apabila server ini belum aktif.
:::

Jalankan masing-masing perintah berikut pada terminal yang terpisah sesuai dengan situs web yang ingin kamu jalankan:

| Situs  | Cmd                    |
| ------ | ---------------------- |
| AYIM   | `npm run ayim-dev`     |
| Closed | `npm run closed-dev`\* |
| MCA    | `npm run mca-dev`      |
| Utama  | `npm run main-dev`     |
| Open   | `npm run open-dev`     |

\*Saat ini belum tersedia

Situs web lokalmu akan dapat ditemui pada alamat URL yang kamu tentukan pada [Referensi Config](../../../misc/reference/config).
