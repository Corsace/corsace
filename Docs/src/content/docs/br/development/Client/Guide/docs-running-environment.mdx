## Valores de Configuração Relevantes

Com base nos valores de configuração fornecidos para `config.docs.host` e `config.docs.port` do site, você deve ser capaz de acessar o site em `http://host:port`

Por exemplo, `http://localhost:10000` se o seu `config.docs.host` for `localhost` e o sua `config.docs.port` for `10000`.

`config.docs.publicUrl` deve ser equivalente à combinação de `config.docs.host` e `config.docs.port`. No exemplo acima, isso seria `http://localhost:10000`.

Consulte a [Referência de Configuração](../../../misc/reference/config) para obter mais informações.

## Executando o Ambiente

Execute o seguinte comando para iniciar o site no modo de desenvolvimento:

```bash
npm run docs-dev
```

O site agora deve estar disponível na URL que você configurou na [Referência de Configuração](../../../misc/reference/config).
