Essa documentação provem informação em como criar, rodar e gerenciar um torneio usando os serviços da Corsace.

## Pré-requisitos

- Tenha certeza de que você está autenticado no [site da Corsace](https://corsace.io) ao menos via osu! ou discord antes de continuar.
- Se você possui um servidor de discord para o torneio, verifique se você não está (perto do) no limite de 250 cargos no servidor.
